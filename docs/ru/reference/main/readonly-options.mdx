# Параметры только для чтения

После инициализации календаря, создаются дополнительные параметры, доступные только для чтения.
Эти параметры содержат информацию о текущем состоянии календаря, которая автоматически обновляется при взаимодействии с ним.

<Info>
  Состояние календаря меняется при каждом взаимодействии с ним, что приводит к обновлению значений этих параметров. Именно поэтому они доступны только для чтения.
</Info>

## context.mainElement

`Type: HTMLElement`

```js
const calendar = new VanillaCalendar('#calendar');
console.log(calendar.context.mainElement);
```

Свойство `mainElement` содержит корневой HTML-элемент, в котором был инициализирован календарь.

- - -

## context.originalElement

`Type: HTMLElement`

```js
const calendar = new VanillaCalendar('#calendar');
console.log(calendar.context.originalElement);
```

Свойство `originalElement` содержит исходный корневой HTML-элемент без модификаций, в котором был инициализирован календарь.

- - -

## context.inputElement

`Type: HTMLElement`

```js

console.log(calendar.context.inputElement);
```

Свойство `inputElement` представляет корневой HTML-элемент, который является полем ввода.
