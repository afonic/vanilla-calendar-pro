# Включить или отключить дни

Иногда вам может потребоваться отключить определенные дни, чтобы они не были доступны для выбора. В приведенных ниже примерах показано, как это можно сделать.

<Info>
  Важно понимать, что в приведенных ниже примерах параметры `settings.selected.year` и `settings.selected.month` указаны только для демонстрации определенного месяца.
</Info>
