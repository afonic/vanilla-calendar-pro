# Первый день недели

В календаре используется стандарт **ISO 8601** для определения первого дня недели. Если параметр `settings.iso8601` установлен в `true`, то первым днем недели будет понедельник. В противном случае, если параметр `settings.iso8601` равен `false`, первым днем недели будет воскресенье. Вы можете настроить этот параметр по вашему усмотрению.
