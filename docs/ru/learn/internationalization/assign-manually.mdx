# Ручная локализация

Вы можете ручным образом локализовать календарь, если параметр `locale` не подходит для ваших нужд.
В календаре локализованы только надписи месяцев и дней недели, поэтому вам нужно предоставить соответствующие массивы строк и установить `locale` в `'define'`.

<Info>
  Обратите внимание, что массив недель должен начинаться с воскресенья, это не влияет на **ISO 8601**.
</Info>
